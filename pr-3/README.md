# 📌 Linked List Implementation in C++

## 📖 Project Description

This project is a simple implementation of a **Singly Linked List** in C++ using **Dynamic Memory Allocation**.
It demonstrates basic operations such as insertion, searching, deletion, reversing, and displaying the list.

---

## 🚀 Features

* Insert element at beginning
* Search element by key
* Delete node by value
* Reverse the linked list
* Display linked list
* Memory cleanup using destructor

---

## 🛠️ Technologies Used

* Language: **C++**
* Concept: **Data Structures (Linked List)**
* Memory Handling: **Dynamic Memory Allocation (new/delete)**

---

## 🧠 Concepts Covered
